STM32CubeF7 Firmware Package V1.6.0 / 30-December-2016

Main Changes
  • Official release of the STM32CubeF7 to add the support of STM32F722xx, STM32F723xx, STM32F732xx and STM32F733xx devices.
  • Add the support of the Low Layer(LL) drivers
  • Update Middleware versions for: FreeRTOS, LibJPEG, StemWin, LwIP and FatFS
  • Introduce the support of mbedTLS Middleware.
  • Full set of examples, applications and demonstrations running on STM32F722ZE-Nucleo and STM32F723E-Discovery
  • Update almost middleware applications to be aligned with new middleware versions
